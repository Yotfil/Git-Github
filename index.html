Comandos git

Inicializando repositorio por primera vez

Si es la primera vez que vas a crear un repositorio deberás seguir estos 6 pasos en el orden indicado. Debes asegurarte de estar ubicado en el Git Bash sobre tu carpeta que quieres trabajar. NO QUERRAS INCIALIZAR UN REPOSITORIO EN LA RAIZ DE TU COMPUTADOR

  1. git init
    Este comando inicializará tu repositorio de manera local

  2. git add .
    Con este comando prepararás tus archivos para subir a tu repositorio local Nota: recuerda que hay una variación de este comando que es especificando el nombre/nombres de los archivos que quieres subir.
    ej:
    git add index.html

  3. git commit -m "first commit"
    Este comando subirá a tu respositorio local (En tu computador) los archivos que alistaste en el paso anterior. Recuerda que en las comillas irá un comentario relacionado con el cambio que se sube

  4. git branch -M main
    Este paso solo será necesario si usas Github y lo que hace es cambiar el nombre de la rama de master a main

  5. git remote add origin https://github.com/user/respositorio.git
    En este punto ya hemos acabado nuestro proceso de forma local (En nuestro computador) y pasaremos a conectar nuestro repositorio con la nube.
    Nota: recuerda que la URL (https://github.com/user/respositorio.git) debe ser la tuya propia que te da el Github

  6. git push -u origin main
    Finalmente este paso hará que nuestros archivos locales (en nuestro computador) sean empujados hacia la nube, que en este caso es (Github)
